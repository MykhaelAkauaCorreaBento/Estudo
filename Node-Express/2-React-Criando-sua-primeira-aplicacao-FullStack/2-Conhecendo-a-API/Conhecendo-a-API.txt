Conhecendo a API


Nesta aula vamos apresentar a API de médicos que vamos consumir no site da Clínica de Pediatria.


Conhecendo a API Na aula anterior nós vimos que a aplicação Front-end consome a API de médicos.


exemplo_1



Agora vamos ver em mais detalhes o Back-end, ou seja, a API de médicos que será consumida 
pelo Front-end.



exemplo_2




Para isso, vamos até o Postman consumir a API através da sua URL.




exemplo_3



Veja que como estamos listando dados a API é consumida com o verbo GET, seguindo o padrão REST.



exemplo_4



Ao consumir a API temos retornado os dados médicos no formato JSON.



exemplo_5


Veja que o retorno é um array de objetos; o objeto é composto por 5 campos: Id, nome, telefone, emall e descrição.
O campo id não será exibido no Front-end.


exemplo_6


Conhecer a API vai facilitar a escrita do código na hora de construir o Front-end.
