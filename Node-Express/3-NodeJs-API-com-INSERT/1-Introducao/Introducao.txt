Introdução

Seja bem-vindo ao curso e parabéns por mais um passo na sua Carreira Programador.

Já sabemos criar APIs que se conectam com um banco de dados.

Neste momento, nossas primeiras APIs conseguem listar informações da tabela do banco, através
do método http GET.

Agora vamos à próxima etapa, criando uma API que também insere registros na tabela do banco 
de dados.

Neste curso, vamos aprender a construir APIs que conseguem inserir registros no banco de dados.
A inserção de dados vai nos permitir expandir o tipo de APIs que podemos criar.

que poderá ser utilizada por um site ou app:
