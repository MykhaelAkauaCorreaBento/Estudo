Validacao de dados.txt

Nesta aula vamos falar sobre um importante processo que deve ser feito quando a API recebe dados 
digitados por um usuário: a validação de dados.

A ideia da validação de dados é filtrar as informações enviadas pelos usuários antes de enviá-las
 ao banco de dados.

Porém, antes de começar, vamos ver como está o código da nossa API até aqui.
