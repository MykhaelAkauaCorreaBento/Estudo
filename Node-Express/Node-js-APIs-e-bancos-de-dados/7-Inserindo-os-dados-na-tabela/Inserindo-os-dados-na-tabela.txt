exemplo_4

Como vimos anteriormente, o campo id será incrementado automaticamente,
por isso não precisamos informá-lo

O comando VALUES indica que o próximo parênteses vai conter os valores que serão
inseridos nos campos.

exemplo_5

Os valores são inseridos na mesma ordem que os campos aparecem no comando, ou seja:

campeao ='Peñarol'
vice = 'Olimpia'
ano = 1960

exemplo_6

Note que os valores dos campos campeao e vice estão entre aspas, mas o ano não.
As aspas são obrigatórias para dados do tipo string, mas não para dados numéricos.

O sinal de ponto e vírgula (;) ao final de cada comando serve para separar os comandos no
MySQL Workbench.
Desse modo podemos executar todos os comandos em sequência automática, ao invés de ter que
'clicar no raio' uma vez para cada comando.


No próximo tópico veremos como inserir um registro graficamente, sem precisar escrever o
comando INSERT.
