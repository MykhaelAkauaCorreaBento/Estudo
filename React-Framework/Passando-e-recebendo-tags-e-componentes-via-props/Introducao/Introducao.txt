Introdução

Em um projeto criado em React é comum a criação e a reutilização de um componente. Através de props conseguimos inserir tags e componentes dentro de outro componente.

Tags e componentes podem ser passados via props para dentro de um componente.
Ao fazer isso, o componente que recebe esses elementos poderá exibi-los junto ao seu código.

import estilos from './page.module.css'
import Imagem09 from '/public/Imagem09.jpeg';
import BoxMensagem from './componentes/BoxMensagem';
import Image from 'next/image';

export default function Home () {
  return (
    <div className={estilos.conteudo}>
      <h1>Minhas mensagens</h1>

      <div className={estilos.mensagens_box}>
        <BoxMensagem>
          <p>Lembra aquele produto que você estava procurando?</p>
          <p>Ele abaixou o preço!! Corre que dá tempo de comprar.</p>
          <a href="https://www.americanas.com.br/1818873801">
            Ver produto
          </a>
        </BoxMensagem>

        <BoxMensagem>
          <p>Separamos esta foto para você.</p>
          <Image src={Imagem09} alt='Água'/>
        </BoxMensagem>
      </div>
    </div>
  )
}

Dessa forma, conseguimos ter um componente com parte do seu conteúdo dinâmico.
Além de valores como string, boolean, number e function conseguimos enviar para um componente também tags e outros componentes via props. 

É através de props que um componente recebe textos, números, booleanos, funções, tags e componentes
