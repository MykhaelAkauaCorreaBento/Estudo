O Array.map é uma função nativa do JavaScript, o que significa que ela não precisa de importação,
e pode ser usada diretamente.

Exemplo de código de utilização do Array.map no React:

jogosExclusivos.map((jogo) => {
   return(
     <ItemJogos key = { jogo.id } nome= { jogo.nome } plataforma = { jogo.plataforma }/>
   )
});

(------------------------------A propriedade key------------------------------)

Agora conheceremos uma importante propriedade do React que deve ser usada junto
do Array.map, a propriedade key.

jogosExclusivos.map((jogo, index) => (
   <ItemJogos key = { index } nome = { jogo.nome } plataforma = { jogo.plataforma }/>
 ));

A key é uma propriedade nativa do React, que ajuda a identificar itens de um array
percorrido pelo Array.map.

Array.map no React
Utilizando a propriedade Key

Key é uma propriedade nativa do React que, obrigatoriamente, deve ser utilizada com o Array.map.

jogosExclusivos.map((jogo, index) => (
   <ItemJogos key = { index } nome = { jogo.nome } plataforma = { jogo.plataforma }/>
 ));

A key define um ID para os componentes criados. Neste exemplo, cada componente ItemJogos
terá um ID único.

A key é necessária para o React conseguir diferenciar componentes iguais criados
atravesdoArray.map.

exemplo_6.png

Veja que, com base na coleção, cada componente ItemJogos criado receberá um ID da prop key.


A propriedade key não precisa ser consumida dentro do componente ItemJogos.

exemplo_7.png


Ao trabalhar com componentes dinâmicos, é obrigatório que o mesmo possua um ID (key).
Caso contrário, o código React apresentará o seguinte erro.

exemplo_8.png

Existe também uma forma simples de passar um valor para key. Você pode usar,
no Array.map, o parâmetro index.

exemplo_9.png

O index retorna o índice de cada item do array. Com isso, você consegue passar um valor de
ID automático.

Componente ItemJogos:
import Image from "next/image";
import estilos from './ItemJogos.module.css';
import xboxLogo from '/public/xbox.png';
import playstationLogo from '/public/playstation.jpg';
import nintendoLogo from '/public/nintendo.jpg';

export default function ItemJogos({ nome, plataforma }) {
  return(
    <div className={estilos.card}>
      <figure>
        <Image
          src={plataforma === "xbox"
              ? xboxLogo
              : plataforma === "playstation"
              ? playstationLogo
              : nintendoLogo}
          alt="logo"
        />
      </figure>
      <div>
        <p></p>
      </div>
    </div>
  )
}


Componente ListaDeJogosExclusivo mostrando o uso do Array.map:
import Image from "next/image";
import estilos from "./page.module.css";
import Lupa from "/public/lupa.png";
import ItemJogos from "./componentes/ItemJogos";
import { jogosExclusivos } from "./dados/banco";

export default function Home() {
  return (
    <div className={estilos.container_principal}>
      <h2>Lista de Jogos Exclusivos</h2>
      <div className={estilos.container_input}>
        <Image src={ Lupa } alt="icone" />
        <input type="text" placeholder="Pesquisar por um jogo" />
      </div>
      <div className={estilos.container_cards}>
        {jogosExclusivos.map((jogo) => (
          <ItemJogos key={jogo.id} nome={jogo.nome} plataforma={jogo.plataforma}/>
        ))}
      </div>
    </div>
  );
}
