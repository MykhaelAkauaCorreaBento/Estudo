O App Router é o sistema de rotas utilizado no React para implementar a navegação entre
páginas do mesmo site em uma aplicação.
Podemos dizer que o App Router é o responsável por criar as rotas de cada página da aplicação.

exemplo:

Ao criar uma aplicação React que utiliza o App Router teremosque utiliza o App Router teremos
uma nova estrutura de pastas e arquivos.


Comparado a uma aplicação que não utiliza o App Router, a nova estrutura criada apresenta
algumas diferenças.

exemplo:
foto7.jpeg

Com a App Router, a pasta app substitui a pasta pages.
exemplo:
foto8.jpeg


A pasta app será responsável por armazenar os arquivos padrão e também todas as
páginas da aplicação.
exemplo:
foto9.jpeg


A pasta styles não existe mais e com isso os arquivos de estilos são armazenados diretamente
na pasta app.

exemplo:
foto10.jpeg


Outra mudança é a substituição do arquivo index.js pelo arquivo page.js, que agora será
o novo arquivo principal da aplicação.

exemplo: 
foto11.jpg


Devido à mudança de arquivo index.js para page.js, o arquivo Home.module.css agora
será page.module.css.

exemplo: 
foto12.jpg


Além de ser o novo arquivo principal, o page.js também será o componente Home e a página principal da aplicação.


exemplo:
