Aplicando classes CSS no React.js


A importação de Home.module.css é nomeada como 'styles'.

import styles from "@/styles/Home.module.css";

styles = Nome definido para o arquivo Home.module.css




Esse nome será utilizado para aplicar o CSS no código React


No caminho de pastas o @/ representa a pasta raiz da aplicação.


styles = Nome definido para o arquivo Home.module.css

@ = Simbolo que indica a pasta src


Para tornar a importação do Home.module.css mais amigável, podemos editar o nome da importação.


import estilos from "@/styles/Home.module.css";


styles foi editado para 'estilos'


Agora, podemos aplicar as classes CSS no código React.js.


import Head from "next/head";
import estilos from "@/styles/Home.module.css";

export default function Home() {
  return (
    <div className={estilos.container_princioal}>
      <Head>
        <title>Create Next App</title>
        <meta name="description" content="Generated by create next app" />
        <meta name="viewport" content="width=device-width, initial-scale=1" />
        <link rel="icon" href="/favicon.ico" />
      </Head>
      <div className={estilos.container_texto}>
        <h1>Hello World</h1>
        <p>Parágrafo com React.js</p>
        <p className={estilos.autor}>Criado por Mykhael</p>
      </div>
    </div>
  );
}

Obs.:
Da mesma forma que em páginas HTML e CSS, utilizaremos classes CSS para estilizar páginas React.js. A diferença nesse processo, em relação ao React.js, fica na importação do arquivo de estilo e na aplicação de classes CSS nas tags HTML.
