Criando uma aplicação React.js


Para criar uma aplicação React.js primeiro você precisa abrir o terminal e acessar a pasta onde
você quer salvar o seu projeto.

exemplo:

 C:\Users\Mykhael Bento\Desktop\Estudo\React-Framework\Projetos\Hello-World>

 Com a pasta desejada aberta, basta executar o comando: npx create-next-app

 npx: é comando que executa o Node.js

 create-next-app: Next.js criando a aplicação React.js




 Este comando precisa ser executado todas as vezes que voce for criar uma aplicação React.Js.

 npx create-next-app


 Ao executar o comando, podemos definir um nome para a aplicação.

 Após definir o nome da aplicação, o terminal dará algumas opções de configurações.

 Precisamos indicar 'yes' apenas para a criação da pasta 'src', pois ela conterá todos os aquivos iniciais padrão da aplicação.

 Por fim, ao concluir o processo de configuração, a aplicação será criada com sucesso.
