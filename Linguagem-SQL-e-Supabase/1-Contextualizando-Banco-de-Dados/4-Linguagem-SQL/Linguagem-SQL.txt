Linguagem-SQL

Nesta quarta e última aula do primeiro curso, o foco se volta para a Linguagem SQL (Structured Query Language), a 
interface universal para comunicação com bancos de dados relacionais. Após entendermos o que são bancos de dados e

O objetivo é desmistificar o SQL, apresentando-o como uma linguagem padronizada, intuitiva e fundamental para 
qualquer profissional que trabalhe com desenvolvimento de sistemas. A aula demonstra os comandos mais básicos e 
estabelece a base para o aprofundamento que ocorrerá nos próximos cursos da formação.


(----------------------------O Que é SQL (Structured Query Language)?----------------------------)

SQL, ou Linguagem de Consulta Estruturada, é a linguagem padrão utilizada para gerenciar e manipular bancos de dados 
relacionais. Independentemente do sistema de banco de dados escolhido  seja PostgreSQL, MySQL, SQL Server ou Oracle 
a base da comunicação será feita através do SQL. Embora possam existir pequenas variações de sintaxe entre os sistemas,
o conhecimento fundamental da linguagem é universalmente aplicável.



Comentário:

linguagem SQL." - Tone

O SQL permite realizar todas as operações necessárias em um banco de dados, desde a criação de sua estrutura até a 
manipulação detalhada dos dados contidos nele.

(----------------------------SQL na Prática: O Comando SELECT----------------------------)


Para ilustrar o poder do SQL, a aula demonstra o uso do comando SELECT, que é a porta de entrada para a extração de
informações do banco de dados.

Exemplo 1: Selecionando Todos os Dados de uma Tabela
Para visualizar todos os serviços cadastrados na tabela servicos, o comando utilizado é:

SELECT * FROM servicos;

SELECT *: Indica que queremos selecionar todas as colunas.
FROM servicos: Especifica que a consulta deve ser feita na tabela servicos.

O resultado é a exibição de todos os registros e todas as colunas da tabela, de forma similar à visualização de uma
planilha completa.





(----------------------------Exemplo 2: Filtrando Dados com a Cláusula WHERE----------------------------)


O verdadeiro poder do SQL começa a aparecer quando filtramos os dados para obter respostas específicas. 
Para buscar apenas o serviço com ID igual a 1, o comando é modificado com a cláusula WHERE:

SELECT * FROM servicos WHERE id = 1;

WHERE id = 1: adiciona uma condição à consulta, instruindo o banco de dados a retornar apenas as linhas onde o valor 
da coluna id é igual a 1.

Essa capacidade de filtrar, ordenar e combinar dados de maneiras complexas é o que torna os bancos de dados 
relacionais muito mais poderosos do que simples planilhas.
