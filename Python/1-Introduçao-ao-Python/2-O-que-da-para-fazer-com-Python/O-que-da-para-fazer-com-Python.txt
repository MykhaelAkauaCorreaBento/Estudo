(---------------------Passo 1: Conhecer as Aplicações Principais de Python---------------------)
Aprendizado: Por que Python se aplica em áreas tão diferentes
Python é uma linguagem de propósito geral, não foi criada para uma área específica, o que a torna uma escolha viável 
em contextos muito diferentes.

No desenvolvimento web, com o suporte de bibliotecas, é possível atuar tanto no back-end quanto no front-end com 
Python, sem precisar alternar entre tecnologias.
Na inteligência artificial, a integração com modelos de IA é direta, sem as barreiras técnicas que outras linguagens 
impõem para o mesmo resultado.
Na análise de dados, grandes volumes de informação são processados com poucas instruções, algo que em outras 
linguagens demandaria muito mais código.
Na automação, a interação nativa com arquivos, sistemas e ferramentas externas torna a criação de scripts mais 
simples e direta do que em outras linguagens.

(---------------------Passo 2: Desenvolvimento Web---------------------)
Aprendizado: Python no back-end
No desenvolvimento web, o back-end é a parte responsável pela lógica que acontece no servidor, processar informações
e responder ao que o usuário solicita. É onde Python se destaca.

Com o suporte de frameworks, é possível criar um servidor e definir o que ele retorna com poucas linhas:

from flask import Flask

app = Flask(__name__)

@app.route("/")
def inicio():
    return "Bem-vindo à minha aplicação!"

app.run()
Ao acessar o endereço no navegador, o servidor responde com a mensagem definida. Poucas linhas são suficientes para 
colocar um servidor no ar.

Aprendizado: Python além do back-end
Python não se limita ao back-end. Com frameworks como o Django, é possível atuar também no front-end, construindo a 
interface visual dentro do mesmo projeto e na mesma linguagem.

Essa abordagem permite o desenvolvimento fullstack sem precisar trocar de linguagem. A versão web do Instagram, por 
exemplo, utiliza o Django dessa forma.

(---------------------Passo 3: Automação com Python---------------------)
Aprendizado: Automatize tudo com poucas linhas
O python permite automatizar tarefas que seriam executadas manualmente, acessar sistemas, coletar informações, 
processar arquivos, transformando um processo repetitivo em algo que roda sozinho.

Um exemplo prático é o web scraping: a extração automática de informações de páginas web. Com a biblioteca 
BeautifulSoup, é possível acessar o conteúdo de uma página e extrair dados específicos com poucas linhas

Veja o exemplo abaixo, que extrai o título e todos os links de uma página:

import requests
from bs4 import BeautifulSoup

pagina = requests.get("https://exemplo.com")
soup = BeautifulSoup(pagina.content, "html.parser")

print("Título:", soup.title.text)

print("\nLinks encontrados:")
for link in soup.find_all("a"):
    print(link.get("href"))
Resultado:

Título: Exemplo de Domínio
Links encontrados:
https://www.iana.org/domains/example
Sem a biblioteca, acessar e interpretar o HTML de uma página exigiria processar o texto manualmente, linha por linha.
Com Python, poucas linhas resolvem.

(---------------------Passo 4: Análise de Dados com Python---------------------)
Aprendizado: Processando dados com Python
Análise de dados é o processo de examinar informações para identificar padrões, tendências e relações que não são 
óbvias à primeira vista. Quanto maior o volume de dados, mais difícil fazer isso manualmente, e é aí que Python entra.

Com o suporte de bibliotecas especializadas, é possível carregar, cruzar e analisar diferentes fontes de dados com 
poucas linhas de código. Veja o exemplo abaixo, que combina dados de temperatura e consumo de energia para identificar
a relação entre os dois:

import pandas as pd

clima = pd.read_csv("temperatura.csv")
energia = pd.read_csv("consumo.csv")

dados = pd.merge(clima, energia, on="mes")
correlacao = dados["temperatura"].corr(dados["consumo"])

print(f"Correlação: {correlacao:.2f}")
print(dados[["mes", "temperatura", "consumo"]].sort_values("consumo", ascending=False))
Resultado:

Correlação: 0.93
mes temperatura consumo
Junho 31.2 5400
Maio 29.8 5100
Julho 28.5 4900
Janeiro 18.3 3200
Fevereiro 19.1 3400
O resultado 0.93 significa que em 93% dos casos, quando a temperatura subiu, o consumo de energia subiu junto. Python 
cruzou dois arquivos de fontes diferentes e chegou a essa conclusão em poucas linhas

(---------------------Passo 5: Inteligência Artificial com Python---------------------)
Aprendizado: Integrando modelos de IA
Na inteligência artificial, o diferencial do Python está na simplicidade de integração, conectar o código a um modelo 
de IA é direto, sem as barreiras técnicas que outras linguagens impõem. Veja o exemplo abaixo, que envia uma pergunta 
a um modelo de IA e exibe a resposta:

from openai import OpenAI
cliente = OpenAI()
resposta = cliente.chat.completions.create(
    model="gpt-4",
    messages=[{"role": "user", "content": "O que é Python?"}]
)
print(resposta.choices[0].message.content)
Resultado:

Python é uma linguagem de programação de alto nível, conhecida pela simplicidade e legibilidade do seu código.
Poucas linhas são suficientes para se comunicar com um modelo de IA e obter uma resposta gerada automaticamente,
a complexidade do modelo fica abstraída pela biblioteca.

(---------------------Aprendizado : Python e MCP = Agentes de IA---------------------)
MCP, Model Context Protocol, é um protocolo que permite conectar modelos de IA a fontes de dados e ferramentas externas
de forma padronizada. É a base sobre a qual agentes de IA são construídos, sistemas capazes de executar tarefas de
forma autônoma, sem intervenção humana a cada passo.

Com Python, essa integração é direta. Veja o exemplo abaixo, que cria um servidor MCP com uma ferramenta simples
usando a biblioteca mcp:

from mcp.server.fastmcp import FastMCP

mcp = FastMCP("meu_servidor")

@mcp.tool()
def somar(a: int, b: int) -> int:
    return a + b

mcp.run()
Com poucas linhas, um modelo de IA passa a ter acesso a uma ferramenta externa, nesse caso, uma função de soma.
O modelo pode chamar essa ferramenta automaticamente durante uma conversa, sem que o usuário precise intervir.

Essa capacidade coloca Python no centro de uma das tendências mais relevantes da inteligência artificial atual.

(---------------------Aprendizado: Python e modelos LLM---------------------)
Além de integrar modelos de IA via API, Python também permite rodar modelos de linguagem, os chamados LLMs, 
diretamente na máquina local. Com a biblioteca ollama, é possível executar modelos como o LLaMA sem depender de 
serviços externos.

Veja o exemplo abaixo, que envia uma pergunta a um modelo LLM rodando localmente:

import ollama

resposta = ollama.chat(
    model="llama3",
    messages=[{"role": "user", "content": "O que é Python?"}]
)

print(resposta["message"]["content"])
Resultado:

Python é uma linguagem de programação de alto nível, conhecida pela simplicidade e legibilidade do seu código.
Sem entrar nos detalhes, perceba que a estrutura do código é muito semelhante à integração com a OpenAI, a diferença é 
que o modelo roda localmente, sem enviar dados para servidores externos.

Para quem quiser ir além, Python também é a linguagem central no ajuste fino de modelos, processo em que um LLM 
existente é treinado com dados específicos para um domínio. Esse é um território mais avançado, mas Python é a 
porta de entrada para ele.
