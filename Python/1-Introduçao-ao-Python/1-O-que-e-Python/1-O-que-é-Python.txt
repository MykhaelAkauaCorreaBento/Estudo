(------------------------Passo 1: Entender o Conceito Fundamental------------------------)
Aprendizado: O que é Python?
Python é uma linguagem de programação de propósito geral. Sendo possível criar diferentes tipos de aplicações: desde 
websites até ferramentas de automação, análise de dados e sistemas de inteligência artificial.

Se destaca pelo fácil aprendizado e utilização. Sua sintaxe foi projetada para ser clara e legível, permitindo que
programadores escrevam códigos de forma mais natural e intuitiva.

Essa abordagem torna Python uma das linguagens mais populares do mundo.

Aprendizado: Por que Python é Popular?
Sua popularidade está associada à resolução de problemas complexos de forma simples e direta. Além disso, possui uma
comunidade ativa e engajada. Isso significa que existem muitos recursos, bibliotecas prontas para uso.

(------------------------Passo 2: Ver Python em Ação------------------------)
Aprendizado: Exemplo de Código em Python
Para entender como Python funciona, observe este exemplo simples:

# Imprimindo uma mensagem
print("Olá, Python!")

# Criando variáveis
nome = "Maria"
idade = 25

# Usando estrutura condicional
if idade >= 18:
    print(f" é maior de idade")
else:
    print(f" é menor de idade")
Este código demonstra como Python permite escrever instruções de forma clara e direta.

Aprendizado: Python comparado a outras linguagens
O mesmo exemplo em Java exigiria muito mais estrutura antes de chegar ao resultado:

public class Main {
    public static void main(String[] args) {
        String nome = "Maria";
        int idade = 25;
        if (idade >= 18) {
            System.out.println(nome + " é maior de idade");
        } else {
            System.out.println(nome + " é menor de idade");
        }
   }
}
O resultado é o mesmo, mas Java exige declarar a classe, o método principal e os tipos de cada variável antes de 
chegar à lógica. Em Python, você vai direto ao ponto.

(------------------------Passo 3: Bibliotecas------------------------)
Aprendizado: Menos código, mais solução
Python se destaca de outras linguagens pelo ecossistema de bibliotecas disponíveis, especialmente nas áreas de automação,
 ferramentas matemáticas e científicas e inteligência artificial. Nessas áreas, Python é referência.

Para ilustrar, veja como a biblioteca matplotlib pode ser usada para gerar um gráfico com a evolução da temperatura no
mês de Agosto:

import matplotlib.pyplot as plt

dias = list(range(1, 20))
temperaturas = [25.5, 33.9, 24.0, 28.3, 27.7, 27.3, 29.0, 21.0, 21.2, 32.7, 24.8, 25.1, 27.8, 36.1, 22.1, 24.6, 21.1, 36.2, 27.0]

plt.plot(dias, temperaturas)
plt.title("Daily Temperatures in August")
plt.xlabel("Day")
plt.ylabel("Temperature (°C)")
plt.show()
Resultado:

Um gráfico de linha exibindo a evolução da temperatura no mês de Agosto.

gráfico de linha
Implementar esse gráfico do zero, sem biblioteca, exigiria calcular proporções, desenhar eixos, posicionar rótulos e
renderizar a imagem manualmente. Com matplotlib, seis linhas resolvem.

(------------------------Aprendizado 2: Bibliotecas em projetos reais------------------------)
Esse ganho se multiplica em projetos mais complexos. Na área de inteligência artificial, por exemplo, é possível 
criar uma integração com um modelo de IA e obter respostas geradas automaticamente com poucas linhas. Veja o exemplo:

from openai import OpenAI

cliente = OpenAI()
resposta = cliente.chat.completions.create(
    model="gpt-4",
    messages=[{"role": "user", "content": "O que é Python?"}]
)

print(resposta.choices[0].message.content)
Resultado:

Python é uma linguagem de programação de alto nível, conhecida pela simplicidade e legibilidade do seu código.

Sem entrar nos detalhes do código, perceba que poucas linhas são suficientes para se comunicar com um modelo de 
inteligência artificial e obter uma resposta gerada automaticamente, algo que poderia parecer extremamente complexo,
resolvido de forma direta.
